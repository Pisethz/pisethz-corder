## Getting Started

### Prerequisites
- Node.js (v18 or newer recommended)
- npm

### Install
```bash
npm install
```

### Run Locally
```bash
npm run dev
```

### Build for Production
```bash
npm run build
```

### Deploy to GitHub Pages
```bash
npm install --save-dev gh-pages
```
Add to `package.json` scripts:
```
"predeploy": "npm run build",
"deploy": "gh-pages -d dist"
```
Then run:
```bash
npm run deploy
```
